Les pilliers d'un soldat

Fondamentaux :

Les fondamentaux sont néccessaire a tout soldats voulant accéder a l'institut, meme si déja maitrser le candidat devra obligatoire suivre tout les cours fondamentaux. Les cours seront composée de : Langues / Pratiques / théroies

Langues : Anglais, Russe, Chinois, Arabe (5h de cours sur une semaine)
Théories : Tactiques/Stratégies militaire , mécanique véhicule/armes, Histoire, Géographie, Politique ( 15h de cours sur une semaine)
Pratique fondamentale : Entrainement physique, Entrainement Tir aux armes, mécanique, premier soin (15h de cours sur une semaine)
